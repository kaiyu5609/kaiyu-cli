## 安装下载

- 直接下载：`https://github.com/kaiyu5609/kaiyu-cli/releases`
- 使用`npm`下载：`npm install kaiyu-cli`
- 使用`bower`下载：`bower install kaiyu-cli`（前提保证电脑已安装了`bower`）
- 使用`CDN`：`http://unpkg.com/kaiyu-cli/release/bundle.js`

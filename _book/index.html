安装下载

  • 直接下载：https://github.com/kaiyu5609/kaiyu-cli/releases
  • 使用npm下载：npm install kaiyu-cli
  • 使用bower下载：bower install kaiyu-cli（前提保证电脑已安装了bower）
  • 使用CDN：http://unpkg.com/kaiyu-cli/release/bundle.js

快速使用

一、初始化新的项目

  • 安装成功之后，进入您的工作空间目录，如：workspaces目录下（或者其他您已创建好的目录）
  • 使用Ctrl + R来打开cmd命令窗口，执行以下命令来初始化您的项目，<project-name>请填写您要初始化的项目名称
kaiyu init <project-name>

二、创建新的项目模块

  • 项目创建好了，那么下一步，我们可以在项目下创建某一个模块
  • 在cmd命令窗口，使用命令cd ./<project-name>/src/module/，进入您项目下的module目录
  • 执行下面的命令来创建一个模块，<module-name>请填写您要创建的模块名称
kaiyu module <module-name>
